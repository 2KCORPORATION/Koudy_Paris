## Leçons apprises

### Technique 1xBet
- Le betslip se vide via `.bet-slip__delete` etc.
- Le code coupon se génère via `.coupon-loader__button` (bouton Enregistrer)
- Le code apparaît dans `.coupon-loader input` (premier input du loader)
- Utiliser `document.querySelector('a[href*="slug"]').closest('li')` pour cliquer les cotes
- Les aria-label des boutons : "V1", "V2", "229.5 Plus de", "225.5 Moins de" etc.
